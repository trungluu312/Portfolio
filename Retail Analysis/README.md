# Retail Analysis

Provide a visual analysis of Retail to show overall sales and effects of Promotional Markdowns on future Sales.

## Main goals

  - Storewise Analysis
  - Departmentwise Analysis
  - Datewise Analysis
  - Promotional Markdown Analysis
